# A113

This is a code bundle for A113. The original project is available at https://www.figma.com/design/U0rF5veAPI1s2UOZuOOfO2/A113.

## Running the code

Run `npm i` to install dependencies.

Frontend only:
- `npm run dev`

Backend API only:
- `npm run dev:server`

Frontend + backend together:
- `npm run dev:full`

API details: [server/README.md](/d:/А113/server/README.md)

## Lab 7 tests

Frontend tests only:
- `npm run test:frontend`

Backend integration tests only:
- `npm run test:backend`

Run both (recommended for final check):
- `npm run test:run`

Run all tests in one Vitest run:
- `npm run test:all`
